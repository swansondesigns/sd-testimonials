# Swanson Designs Website

Official website for Swanson Designs built with Tailwind CSS 4.

**Features:**
- Modern design with custom typography system
- Tailwind CSS 4 with design tokens
- Custom brand colors and fonts from Figma
- Performance optimized

**Project Structure:**
```
sd-website/
├── src/css/            # Source CSS files
│   ├── tailwind.css    # Main input file with @theme tokens
│   └── safelist.css    # Safelist for dynamic classes
├── dist/css/           # Output directory for compiled CSS
├── index.html          # Main HTML file
└── README.md           # This file
```
